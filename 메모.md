## 하나의 구만 사용하여 분석/예측 진행하기
- 데이터가 많은 지역만 사용할지 또는 NaN 값이 없는 지역만 할지 등등의 기준을 정하기

## 여러 csv를 병합하기
- '구'라는 컬럼을 생성하여 어떤 구의 데이터인지를 확인할 수 있도록 함
- feature 중요도 시각화 등을 할 때는 이 데이터는 제외하고 사용
- 여러 값을 주고 어떤 구인지 예측할 때는 랜덤 포레스트를 이용하여 분류하도록 함
- 구 이외에 그냥 회귀쪽 예측이라고 하면 다른 알고리즘과 성능을 비교한 뒤에 진행해보도록
